# Java ATM Konsol Uygulaması

Bu proje, kullanıcıların konsol üzerinden bir ATM sistemini simüle edebileceği basit bir **Java** uygulamasıdır.  
Kullanıcılar hesap oluşturabilir, giriş yapabilir, bakiyelerini görüntüleyebilir, para yatırabilir ve para çekebilirler.

---

## Özellikler

- **Hesap Oluşturma:** Yeni kullanıcı hesabı oluşturabilirsiniz.  
- **Giriş Yapma:** Daha önce oluşturduğunuz hesapla giriş yapabilirsiniz.  
- **Bakiye Görüntüleme:** Mevcut bakiyenizi anlık olarak görebilirsiniz.  
- **Para Yatırma:** Hesabınıza para ekleyebilirsiniz.  
- **Para Çekme:** Bakiyeniz yeterliyse para çekebilirsiniz.  
- **Çıkış:** Oturumunuzu sonlandırabilirsiniz.

---

## Kullanılan Teknolojiler

- **Java**
